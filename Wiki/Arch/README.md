## Соглашение

- `viewBox="0 0 680 H"`, `width="100%"` — масштабируется под контейнер.
- Цвета заданы явными hex (фон белый) — читаемо в любой теме просмотрщика.
- Один файл = одна диаграмма, имя файла совпадает с именем заметки.
